3. Selection & Indexing Most-Used Feature

Column Selection

df["name"]                  # single column → Series
df[["name", "age"]]          # multiple columns → DataFrame
df.name                      # attribute access (only valid Python names)

Row Selection — loc vs iloc

# .loc — LABEL-based (inclusive end)
df.loc[0]                            # row with label 0
df.loc[0:2]                          # labels 0,1,2 (INCLUSIVE)
df.loc[[0, 2, 5]]                    # specific rows
df.loc[df["age"] > 25]               # boolean mask
df.loc[0, "name"]                    # single cell
df.loc[0:2, ["name", "age"]]         # rows + columns
df.loc[:, "age"]                     # all rows, one column

# .iloc — POSITION-based (exclusive end, like Python slicing)
df.iloc[0]                           # first row
df.iloc[0:3]                         # first 3 rows (EXCLUSIVE)
df.iloc[-1]                          # last row
df.iloc[[0, 2, 5]]                   # specific positions
df.iloc[0, 1]                        # first row, second column
df.iloc[:, 0:3]                      # all rows, first 3 cols
Key gotcha: .loc is INCLUSIVE of end (df.loc[0:2] → 3 rows). .iloc is EXCLUSIVE (df.iloc[0:2] → 2 rows).

Boolean Indexing

df[df["age"] > 25]
df[(df["age"] > 25) & (df["city"] == "BLR")]       # AND with &
df[(df["age"] < 18) | (df["age"] > 65)]            # OR with |
df[~(df["status"] == "inactive")]                  # NOT with ~

# Multiple values
df[df["city"].isin(["BLR", "HYD"])]
df[~df["city"].isin(["BLR", "HYD"])]

# String matching
df[df["email"].str.contains("@gmail")]
df[df["name"].str.startswith("A")]
df[df["name"].str.match(r"^[A-M]")]                # regex

# Between
df[df["age"].between(25, 35)]                      # inclusive both ends

# Null filtering
df[df["email"].isna()]
df[df["email"].notna()]

query() — SQL-like Filtering

df.query("age > 25 and city == 'BLR'")
df.query("age in [25, 30, 35]")
df.query("name.str.startswith('A')", engine="python")

# Reference Python variables with @
min_age = 25
df.query("age > @min_age")

at / iat — Fastest single-cell access

df.at[0, "name"]            # label-based, single value
df.iat[0, 1]                # position-based, single value
# Much faster than loc/iloc for one cell

Setting Values (Avoid SettingWithCopyWarning)

# GOOD: use .loc for setting
df.loc[df["age"] > 30, "category"] = "senior"
df.loc[0, "name"] = "New Name"

# BAD: chained assignment (warning + may not work)
df[df["age"] > 30]["category"] = "senior"   # ❌

# Creating new columns
df["full_name"] = df["first"] + " " + df["last"]
df["doubled_age"] = df["age"] * 2
df = df.assign(new_col=lambda x: x["age"] * 2)   # chainable

4. Data Cleaning 80% of DE Time

Handling Missing Data

# Check
df.isna().sum()                      # nulls per column
df.isna().sum().sum()                # total nulls
df.isna().mean() * 100               # % nulls per column

# Drop
df.dropna()                                       # any null in row
df.dropna(how="all")                              # all nulls in row
df.dropna(subset=["email", "phone"])              # nulls in specific cols
df.dropna(thresh=3)                               # need at least 3 non-null
df.dropna(axis=1)                                 # drop columns with any null

# Fill
df.fillna(0)
df.fillna({"age": 0, "name": "Unknown"})          # different per column
df.fillna(method="ffill")                         # forward fill
df.fillna(method="bfill")                         # backward fill
df["age"].fillna(df["age"].median())              # impute with median

# Interpolate (for numeric series)
df["temperature"].interpolate(method="linear")

Duplicates

df.duplicated()                                   # boolean mask
df.duplicated(subset=["email"])                   # consider specific cols
df[df.duplicated(keep=False)]                     # show all duplicate rows

df.drop_duplicates()                              # remove dups
df.drop_duplicates(subset=["email"], keep="last") # keep last occurrence

Data Types

df.dtypes
df["age"] = df["age"].astype("int32")             # downcast int
df["price"] = df["price"].astype("float32")
df["category"] = df["category"].astype("category")   # huge memory savings!
df["date"] = pd.to_datetime(df["date"])
df["date"] = pd.to_datetime(df["date"], format="%Y-%m-%d", errors="coerce")
# errors='coerce' → bad values become NaT

# Safe numeric conversion
df["amount"] = pd.to_numeric(df["amount"], errors="coerce")

# Convert dtypes intelligently
df = df.convert_dtypes()                          # uses nullable types

String Operations

df["name"].str.upper()
df["name"].str.lower()
df["name"].str.title()
df["name"].str.strip()
df["name"].str.len()
df["email"].str.contains("@gmail")
df["email"].str.startswith("admin")
df["email"].str.replace("@old.com", "@new.com")
df["full_name"].str.split(" ", expand=True)       # returns DataFrame
df["name"].str[0:3]                                # slice strings
df["text"].str.extract(r"(\d+)")                  # regex group
df["text"].str.findall(r"\d+")                    # all matches
df["category"].str.cat(df["sub"], sep="-")        # concatenate columns

Renaming & Transforming

# Rename columns
df = df.rename(columns={"old_name": "new_name"})
df.columns = ["a", "b", "c"]                      # all at once
df.columns = df.columns.str.lower().str.replace(" ", "_")   # clean

# Drop columns/rows
df = df.drop(columns=["temp_col"])
df = df.drop(columns=["a", "b"])
df = df.drop(index=[0, 5, 10])

# Replace values
df["status"] = df["status"].replace({"A": "Active", "I": "Inactive"})
df = df.replace({"N/A": np.nan, "missing": np.nan})

# Map (Series only)
df["status_label"] = df["status"].map({"A": "Active", "I": "Inactive"})

# Apply — element-wise or row/col-wise (slower than vectorized!)
df["age_group"] = df["age"].apply(lambda x: "adult" if x >= 18 else "minor")
df.apply(lambda row: row["price"] * row["qty"], axis=1)   # across rows

# applymap (deprecated → use map)
# Better: vectorize when possible

# Cut / qcut (binning)
df["age_bucket"] = pd.cut(df["age"], bins=[0, 18, 30, 60, 100],
                          labels=["child", "young", "adult", "senior"])
df["income_quartile"] = pd.qcut(df["income"], q=4, labels=["Q1","Q2","Q3","Q4"])

Validation Patterns

# Detect outliers (IQR method)
Q1 = df["price"].quantile(0.25)
Q3 = df["price"].quantile(0.75)
IQR = Q3 - Q1
outliers = df[(df["price"] < Q1 - 1.5*IQR) | (df["price"] > Q3 + 1.5*IQR)]

# Range check
assert df["age"].between(0, 150).all(), "Invalid ages found"

# Unique constraint
assert df["email"].is_unique, "Duplicate emails!"

# Use pandera for production data quality

import pandera as pa

    "email": pa.Column(str, checks=pa.Check.str_matches(r"^[\w\.]+@\w+\.\w+$")),
})
schema.validate(df)

5. GroupBy & Aggregation

Basic GroupBy

df.groupby("region")["sales"].sum()
df.groupby("region")["sales"].agg(["sum", "mean", "count"])

# Multiple group keys
df.groupby(["region", "product"])["sales"].sum()

# Multiple aggregations per column
df.groupby("region").agg({
    "sales": ["sum", "mean"],
    "quantity": "sum",
    "customer_id": "nunique"
})

# Named aggregations (cleaner output)
df.groupby("region").agg(
    total_sales=("sales", "sum"),
    avg_sales=("sales", "mean"),
    unique_customers=("customer_id", "nunique"),
    max_order=("sales", "max"),
)

Common Aggregations

MethodWhat
sum()Sum
mean() / median()Average / Median
min() / max()Min / Max
count()Non-null count
size()Total rows (incl. nulls)
nunique()Unique value count
std() / var()Standard deviation / Variance
first() / last()First / Last value
quantile(0.95)Percentile
describe()Multiple stats at once

transform vs apply vs agg

# agg: returns ONE value per group → reduces rows
df.groupby("region")["sales"].agg("sum")

# transform: returns SAME shape as input → broadcast back to original rows
df["sales_pct_of_region"] = df["sales"] / df.groupby("region")["sales"].transform("sum")
df["region_mean"] = df.groupby("region")["sales"].transform("mean")
df["zscore"] = df.groupby("region")["sales"].transform(lambda x: (x - x.mean()) / x.std())

# apply: most flexible, returns anything
def top_n(group, n=3):
    return group.nlargest(n, "sales")
df.groupby("region").apply(top_n, n=5)

Filter Groups

# Keep only groups with > 10 rows
df.groupby("region").filter(lambda g: len(g) > 10)

# Keep groups where total sales > 100000
df.groupby("customer_id").filter(lambda g: g["sales"].sum() > 100000)

Iterate Over Groups

for name, group_df in df.groupby("region"):
    print(name, group_df.shape)
    # process each group

Cross-tabulation / Pivot

# Counts by two dimensions
pd.crosstab(df["region"], df["product"])
pd.crosstab(df["region"], df["product"], values=df["sales"], aggfunc="sum")
pd.crosstab(df["region"], df["product"], normalize="index")   # row %s

# pivot_table (same as Excel pivot)
df.pivot_table(
    index="region",
    columns="product",
    values="sales",
    aggfunc="sum",
    fill_value=0,
    margins=True,                # add totals
    margins_name="Total"
)

Window-like Operations (Ranking, Top-N per Group)

# Rank within group
df["rank"] = df.groupby("region")["sales"].rank(ascending=False, method="dense")

# Top 3 per group
df.sort_values(["region", "sales"], ascending=[True, False]) \
  .groupby("region").head(3)

# Cumulative sum within group
df["cumsum"] = df.groupby("region")["sales"].cumsum()

# Running mean (rolling)
df["rolling_avg"] = df.groupby("region")["sales"].rolling(7).mean().values

# Shift (LAG/LEAD)
df["prev_sale"] = df.groupby("customer_id")["sales"].shift(1)
df["next_sale"] = df.groupby("customer_id")["sales"].shift(-1)

6. Merge, Join & Concat

merge() — SQL-style joins

# Inner join (default)
result = df1.merge(df2, on="customer_id")

# Different column names
result = df1.merge(df2, left_on="cust_id", right_on="id")

# Outer joins
result = df1.merge(df2, on="customer_id", how="left")
result = df1.merge(df2, on="customer_id", how="right")
result = df1.merge(df2, on="customer_id", how="outer")
result = df1.merge(df2, on="customer_id", how="cross")        # cartesian

# Multiple keys
result = df1.merge(df2, on=["customer_id", "date"])

# Suffixes for conflicting columns
result = df1.merge(df2, on="id", suffixes=("_left", "_right"))

# Indicator (where did each row come from?)
result = df1.merge(df2, on="id", how="outer", indicator=True)
# adds _merge column: 'left_only', 'right_only', 'both'

# Anti-join (in df1 NOT in df2)
result = df1.merge(df2, on="id", how="left", indicator=True)
anti = result[result["_merge"] == "left_only"].drop(columns=["_merge"])

join() — Index-based join (less common)

df1.set_index("id").join(df2.set_index("id"), how="inner")

concat() — Stack DataFrames

# Vertical (stack rows) — like UNION ALL
result = pd.concat([df1, df2], ignore_index=True)
result = pd.concat([df1, df2], axis=0)                  # same thing

# Horizontal (combine columns)
result = pd.concat([df1, df2], axis=1)

# Keep only common columns
result = pd.concat([df1, df2], join="inner")

# With keys (creates multi-index)
result = pd.concat([df1, df2], keys=["jan", "feb"])

# Read many files and combine
import glob
files = glob.glob("data/2026-*.csv")
df = pd.concat([pd.read_csv(f) for f in files], ignore_index=True)

merge_asof() — Time-based fuzzy join

# Match each row in left with most recent row in right
trades = pd.DataFrame({"time": [...], "ticker": [...], "price": [...]})
quotes = pd.DataFrame({"time": [...], "ticker": [...], "bid": [...]})

result = pd.merge_asof(
    trades.sort_values("time"),
    quotes.sort_values("time"),
    on="time",
    by="ticker",
    tolerance=pd.Timedelta("2ms"),
    direction="backward"           # 'backward', 'forward', 'nearest'
)

combine_first() — Fill nulls from another DataFrame

# Like COALESCE — use df1 values, fall back to df2 for nulls
result = df1.combine_first(df2)

pivot() — Long to Wide (no aggregation)

df.pivot_table(
    index="region",
    columns="product",
    values="sales",
    aggfunc="sum",                       # or 'mean', 'count', list of funcs

    margins=True

melt() — Wide to Long

# Wide format:
#  date        prod_A  prod_B  prod_C
#  2026-01-01    100     200     150

df.melt(
    id_vars=["date"],                    # keep as is
    value_vars=["prod_A", "prod_B", "prod_C"],
    var_name="product",
    value_name="sales"
)
# Long format:
#  date        product   sales
#  2026-01-01  prod_A    100
#  2026-01-01  prod_B    200
#  ...

stack() / unstack()

# stack: columns → rows (creates multi-index)
df.stack()

# unstack: rows → columns (inverse of stack)
df.unstack()

# Useful with groupby
df.groupby(["region", "product"])["sales"].sum().unstack("product")

explode() — List values to rows

df = pd.DataFrame({"id": [1,2], "tags": [["a","b"], ["c","d","e"]]})

# id   tags
#  1    a
#  1    b
#  2    c
#  2    d
#  2    e

get_dummies() — One-hot encoding

pd.get_dummies(df["category"])
pd.get_dummies(df, columns=["category", "region"], drop_first=True)

wide_to_long() — Complex pattern reshaping

# For columns like q1_sales, q2_sales, q1_profit, q2_profit
pd.wide_to_long(df, stubnames=["sales", "profit"],
                i="region", j="quarter", sep="_", suffix=r"q\d+")

8. Time Series Operations

Datetime Basics

df["date"] = pd.to_datetime(df["date"])

# Extract components
df["year"] = df["date"].dt.year
df["month"] = df["date"].dt.month
df["day"] = df["date"].dt.day
df["weekday"] = df["date"].dt.day_name()
df["quarter"] = df["date"].dt.quarter
df["hour"] = df["date"].dt.hour
df["is_weekend"] = df["date"].dt.weekday >= 5

# Date arithmetic
df["next_week"] = df["date"] + pd.Timedelta(days=7)
df["age_days"] = (pd.Timestamp.now() - df["birth"]).dt.days

# Date offsets (calendar-aware)
df["next_month"] = df["date"] + pd.DateOffset(months=1)
df["bday"] = df["date"] + pd.offsets.BDay(1)        # next business day

Date Range Generation

pd.date_range("2026-01-01", "2026-12-31", freq="D")           # daily
pd.date_range("2026-01-01", periods=12, freq="MS")            # 12 month starts
pd.date_range("2026-01-01", periods=10, freq="W-MON")         # Mondays
pd.date_range("2026-01-01", periods=24, freq="H")             # hourly
pd.bdate_range("2026-01-01", "2026-12-31")                    # business days

# Common freq codes:
# D, W, M, Q, Y / H, T (min), S, MS (month start), BM (biz month end)

Resampling (Time-based GroupBy)

df = df.set_index("date")

# Downsample (e.g., daily → monthly)
monthly = df.resample("M").sum()
monthly = df.resample("M").agg({"sales": "sum", "orders": "count"})

# Upsample (daily → hourly with fill)
hourly = df.resample("H").ffill()

# Custom periods
df.resample("W-MON").sum()              # weekly starting Monday
df.resample("Q").mean()                 # quarterly

# OHLC for stock-like data
df["price"].resample("D").ohlc()        # open, high, low, close

Rolling Windows

df["rolling_7d"] = df["sales"].rolling(window=7).mean()
df["rolling_30d"] = df["sales"].rolling(window=30, min_periods=1).sum()
df["rolling_std"] = df["sales"].rolling(7).std()

# Time-based window (works on DatetimeIndex)
df["sales"].rolling("7D").mean()        # last 7 days regardless of row count

# Expanding (cumulative from start)
df["cumulative_avg"] = df["sales"].expanding().mean()

# Exponentially weighted
df["ewm"] = df["sales"].ewm(span=10).mean()

Shifting (Lag/Lead)

df["prev_day"] = df["sales"].shift(1)              # LAG 1
df["next_day"] = df["sales"].shift(-1)             # LEAD 1
df["week_ago"] = df["sales"].shift(7)
df["day_over_day"] = df["sales"] - df["sales"].shift(1)
df["pct_change"] = df["sales"].pct_change()
df["growth_yoy"] = df["sales"].pct_change(periods=365) * 100

Timezones

df["date"] = df["date"].dt.tz_localize("UTC")
df["date_ist"] = df["date"].dt.tz_convert("Asia/Kolkata")
df["date"] = df["date"].dt.tz_localize(None)        # remove tz

9. Performance & Best Practices

Vectorize, Don't Loop!

# ❌ SLOW (Python loop)
for i, row in df.iterrows():
    df.at[i, "new"] = row["a"] + row["b"]

# ❌ SLOW (apply with lambda)
df["new"] = df.apply(lambda row: row["a"] + row["b"], axis=1)

# ✅ FAST (vectorized — 100-1000x faster!)
df["new"] = df["a"] + df["b"]

# ✅ FAST (numpy where for conditionals)
import numpy as np
df["status"] = np.where(df["age"] >= 18, "adult", "minor")

# ✅ FAST (np.select for multi-condition)
conditions = [df["age"] < 13, df["age"] < 20, df["age"] < 60]
choices = ["child", "teen", "adult"]
df["group"] = np.select(conditions, choices, default="senior")

Optimize Data Types (HUGE Memory Savings)

# Check current memory
df.memory_usage(deep=True).sum() / 1024**2    # MB

# Downcast numeric
df["int_col"] = pd.to_numeric(df["int_col"], downcast="integer")
df["float_col"] = pd.to_numeric(df["float_col"], downcast="float")

# Convert strings to category (huge wins for low-cardinality)
df["country"] = df["country"].astype("category")   # often 90% smaller

# Manual dtype selection
df = df.astype({"id": "int32", "age": "int8", "price": "float32"})

# Read CSV with optimal types from the start
df = pd.read_csv("data.csv", dtype={
    "id": "int32",
    "country": "category",
    "price": "float32",
})

Chunk Processing for Big Files

chunk_iter = pd.read_csv("huge.csv", chunksize=100_000)
results = []
for chunk in chunk_iter:
    chunk = chunk[chunk["amount"] > 0]
    chunk["doubled"] = chunk["amount"] * 2
    results.append(chunk.groupby("region")["amount"].sum())

final = pd.concat(results).groupby(level=0).sum()

Use eval / query for Speed

# For BIG dataframes (millions of rows), eval/query use numexpr internally
df.eval("total = price * qty", inplace=True)
df.query("price > 100 and qty > 5")

Avoid Common Anti-Patterns

# ❌ Growing DataFrames in a loop (O(n²))
df = pd.DataFrame()
for x in items:
    df = pd.concat([df, transform(x)])

# ✅ Collect then concat once
parts = [transform(x) for x in items]
df = pd.concat(parts, ignore_index=True)

# ❌ Chained indexing (warning + slow)
df[df["age"] > 30]["category"] = "senior"

# ✅ Use .loc once
df.loc[df["age"] > 30, "category"] = "senior"

# ❌ apply when vectorization works
df["sqrt"] = df["x"].apply(np.sqrt)

# ✅ Direct vectorized
df["sqrt"] = np.sqrt(df["x"])

Use Parquet, Not CSV

# Parquet vs CSV:
#  - 5-10x smaller files
#  - 10-100x faster reads (columnar)
#  - Schema preserved (no parse_dates needed)
#  - Column pruning (read only what you need)

df.to_parquet("data.parquet", compression="snappy")
df = pd.read_parquet("data.parquet", columns=["id", "amount"])  # only 2 cols

Profile Slow Code

# In Jupyter
%timeit df["a"] + df["b"]
%memit big_operation()              # pip install memory_profiler

# In scripts
import cProfile
cProfile.run("process(df)")

10. Interview Q&A

Q1. Series vs DataFrame?
Series: 1D labeled array (one column). DataFrame: 2D labeled table (multiple Series sharing an index).
Q2. .loc vs .iloc?
.loc: label-based, INCLUSIVE end. .iloc: position-based, EXCLUSIVE end (Python-like slicing).
Q3. apply vs map vs applymap?
map: Series only, element-wise (use dict/Series for mapping). apply: Series (element-wise) or DataFrame (row/col-wise). applymap: deprecated → use DataFrame.map for element-wise on DataFrame.
Q4. merge vs join vs concat?
merge: SQL-style joins on columns or indexes. join: index-based join (convenience). concat: stacks DataFrames vertically or horizontally (like UNION ALL).
Q5. How to handle missing data?
Detect with isna(). Options: drop (dropna), fill with constant/method/statistic (fillna), interpolate, or model-based imputation. Choice depends on % missing and business context.
Q6. How to read a 50GB CSV?
Use chunksize: pd.read_csv("file.csv", chunksize=100000). Or switch to Polars/Dask/Spark. Better: convert to Parquet once, then read columnar.
Q7. transform vs apply vs agg in groupby?
agg: returns one value per group (reduces). transform: returns same shape as input (broadcasts back). apply: most flexible, returns anything.
Q8. How do you optimize memory for a Pandas DataFrame?
Downcast numerics (int64→int8), convert low-cardinality strings to category, drop unused columns, use Parquet for storage, process in chunks.
Q9. What is the SettingWithCopyWarning?
Pandas warns when you set values on what might be a view vs copy. Fix by using .loc for setting in a single operation, or explicitly .copy() the slice.
Q10. Pivot vs Pivot Table?
pivot(): no aggregation, fails on duplicate keys. pivot_table(): with aggregation function, handles duplicates.
Q11. How to deduplicate keeping latest?
df.sort_values("updated_at").drop_duplicates("id", keep="last")
Q12. Difference between merge how='left' and how='outer'?
'left': all rows from left + matching from right. 'outer': all rows from both.
Q13. How to find top 3 rows per group?
df.sort_values(["grp","score"], ascending=[True,False]).groupby("grp").head(3) or use nlargest within apply.
Q14. Pandas vs Spark — when to use which?
Pandas: data fits in memory (~5GB). Spark: distributed, terabytes, cluster available. PySpark also useful when integrating with Hadoop/Delta Lake ecosystem.
Q15. How to implement SCD Type 2 in Pandas?
Compare staging with current. For changed rows: update current's valid_to + is_current=False; insert new row with valid_from=today, is_current=True. Use merge with indicator to find new vs changed.
