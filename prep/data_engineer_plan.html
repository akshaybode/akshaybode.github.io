Month 3 Project: End-to-End Cloud Data Pipeline

Build a complete production-style pipeline:

  • Source: Fetch data from API (e.g., Spotify, IPL stats, stocks) every hour
  • Ingest: Lambda → S3 (raw zone, partitioned by date)
  • Transform: Glue PySpark job → S3 (curated zone, Parquet)
  • Load: Snowpipe auto-loads curated → Snowflake staging tables
  • Model: dbt transforms staging → marts (star schema)
  • Orchestrate: Airflow DAG runs entire pipeline with dependencies
  • Monitor: CloudWatch + Airflow alerts to Slack/email
  • Document: Architecture diagram + README + blog post on Medium

❓ Frequently Asked

Q: Can I skip Kafka?

A: Learn basics only (1 week). 60% of DE roles don't need it day 1. Don't skip if you target product companies (Flipkart, Swiggy etc.).

Q: AWS vs Azure vs GCP — which one?

A: For India market in 2026 — AWS is #1 (60% of jobs), Azure #2 (25%), GCP #3 (15%). Stick with AWS for highest ROI.

Q: Should I learn Databricks instead of Snowflake?

A: Both are growing. Snowflake is easier to start with and more common in BFSI/retail. Databricks is bigger in ML-heavy companies. Learn Snowflake now, add Databricks in month 5-6 on the job.

Q: Do I need Java for DE?

A: Not really. Python is the primary language. Your Java is a bonus (helps debug Spark JVM issues, read source code).

Q: Should I quit my current job to focus full-time?

A: NO. Keep your job for income + safety net. Study 4-5 hrs/day after work. Quit only when you have an offer in hand.

Q: What if I can't get a DE job in 4 months?

A: Realistic timeline is 4-7 months including job search. Don't lose hope. Internal transfers within your current company are also a great path — talk to data teams internally.
